# Деплой Discord бота на Replit

## Пошаговая инструкция

### 1. Создание Replit проекта
1. Зайдите на https://replit.com
2. Нажмите "Create Repl"
3. Выберите "Import from GitHub"
4. Вставьте ссылку на ваш GitHub репозиторий
5. Или создайте новый Python проект и загрузите файлы

### 2. Настройка переменных окружения
1. В Replit откройте вкладку "Secrets" (🔒)
2. Добавьте следующие переменные:
   ```
   DISCORD_TOKEN=ваш_токен_бота
   GUILD_ID=id_вашего_сервера
   OWNER_IDS=id_владельца1,id_владельца2
   PREFIX=!
   LOG_LEVEL=INFO
   DB_PATH=bot.db
   ```

### 3. Запуск бота
1. Нажмите кнопку "Run" (▶️)
2. Бот должен запуститься и подключиться к Discord
3. В консоли увидите "Bot is running!" - это keep-alive сервер

### 4. Настройка Always On (опционально)
1. В Replit Pro можно включить "Always On"
2. Это предотвратит засыпание бота
3. Бесплатная версия: бот может спать после 5 минут неактивности

### 5. Мониторинг
- Следите за логами в консоли Replit
- При ошибках бот автоматически перезапустится
- Keep-alive сервер работает на порту 8080

## Структура файлов для Replit
```
/
├── main.py              # Точка входа
├── keep_alive.py        # Keep-alive сервер
├── .replit              # Конфигурация Replit
├── requirements.txt     # Зависимости
├── .env                 # Переменные окружения (не загружайте!)
└── src/                 # Исходный код бота
    ├── bot.py
    ├── config.py
    ├── cogs/
    └── utils/
```

## Важные замечания
- НЕ загружайте .env файл в Replit
- Используйте Secrets для токенов
- Бот автоматически перезапускается при изменениях
- Keep-alive предотвращает засыпание (частично)

## Troubleshooting
- Если бот не запускается: проверьте Secrets
- Если команды не работают: проверьте права бота
- Если бот спит: включите Always On или используйте UptimeRobot
